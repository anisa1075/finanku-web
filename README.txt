FinankuApp

Deploy Website :
https://finanku-web.vercel.app/

Aplikasi web untuk membantu pengguna mengatur keuangan pribadi, dibangun dengan React + Vite dan Tailwind CSS.

🚀 Fitur Utama
Input & manajemen pemasukan dan pengeluaran
Tampilan responsif (mobile & desktop)
Komponen modern dengan Tailwind CSS
Navigasi mudah & cepat

📂 Struktur Project
finankuApp/
│── public/          # File statis
│── src/             # Source code utama
│   ├── assets/      # File Assets gambar
│   ├── components/  # Komponen UI
│   ├── Pages/       # Halaman aplikasi
│   └── App.jsx      # Root React component
│── index.html       # Entry HTML
│── package.json     # Dependencies & scripts
│── README.txt       # Dokumentasi project

🛠️ Teknologi
React + Vite
Tailwind CSS
(Opsional) Swiper.js, Chart.js / Recharts

⚙️ Cara Menjalankan
1. Download File ZIP
2. Ekstrak File
2. Open File
3. Install dependencies:
npm install
4. Jalankan server development :
npm run dev
5. Buka di browser sesuai alamat yang ditampilkan (biasanya http://localhost:5173/).
